## 1. 设计概述

### 1.1 业务领域
备件管理模块是NESOM系统的物资管理核心，负责新能源设备备件的全生命周期管理。主要业务领域包括：

1. **备件档案管理**：备件分类、属性、编码、技术参数管理
2. **库存管理**：多仓库库存实时管理、库位精细化管理
3. **入库管理**：采购收货、退货入库、调拨入库、生产入库
4. **出库管理**：工单领用、调拨出库、退货出库、报废处理
5. **盘点管理**：定期盘点、循环盘点、差异处理
6. **采购管理**：供应商管理、采购计划、订单跟踪
7. **成本管理**：成本核算、库存估值、呆滞料分析
8. **预警管理**：库存预警、效期预警、呆滞料预警

### 1.2 核心业务流程
1. 备件需求 → 采购计划 → 采购订单 → 供应商送货 → 收货检验 → 入库上架
2. 设备维护 → 工单创建 → 备件需求 → 库存预留 → 领用出库 → 成本核算
3. 库存盘点 → 差异发现 → 差异分析 → 调整审批 → 库存调整 → 流程优化
4. 库存监控 → 预警触发 → 预警通知 → 响应处理 → 预防措施

### 1.3 设计原则
1. **库存准确性**：实时库存数据，事务保障数据一致性
2. **流程标准化**：标准化入库、出库、调拨、盘点流程
3. **成本精细化**：支持多种计价方法，精确成本核算
4. **预警智能化**：基于规则和算法的智能预警
5. **移动端支持**：扫码、拍照、离线操作全面支持
6. **安全合规**：操作可追溯，数据可审计，权限精细化

## 2. 核心业务实体状态机

### 2.1 备件状态机 (Spare Part Status State Machine)

#### 2.1.1 状态定义
| 状态 | 编码 | 描述 | 业务含义 |
|------|------|------|----------|
| 启用 | active | 正常使用状态 | 可采购、可领用、可入库 |
| 停用 | inactive | 暂时停用状态 | 不可采购，可消耗库存 |
| 淘汰 | obsolete | 已淘汰状态 | 不可采购和领用，仅可报废 |

#### 2.1.2 状态转换规则
```
                                     ┌─────────────┐
                                     │   active    │◄───创建备件
                                     └──────┬──────┘
                                            │ 停用
                                            ▼
                                     ┌─────────────┐
                                     │  inactive   │
                                     └──────┬──────┘
                                            │ 启用/淘汰
                                            ▼
                                     ┌─────────────┐
                                     │  obsolete   │
                                     └─────────────┘
```

#### 2.1.3 状态转换约束
| 转换 | 原状态 | 新状态 | 触发条件 | 必填信息 | 权限要求 |
|------|--------|--------|----------|----------|----------|
| 启用 | inactive | active | 备件恢复使用 | 启用原因 | 备件管理员 |
| 停用 | active | inactive | 备件暂时停用 | 停用原因 | 备件管理员 |
| 淘汰 | active/inactive | obsolete | 备件永久淘汰 | 淘汰原因、处理方案 | 备件管理员+审批 |

### 2.3 库存事务状态机 (Inventory Transaction State Machine)

#### 2.3.1 状态定义
| 状态 | 编码 | 描述 | 业务含义 |
|------|------|------|----------|
| 待处理 | pending | 事务创建，等待执行 | 可取消 |
| 处理中 | processing | 正在执行库存操作 | 不可取消 |
| 已完成 | completed | 库存操作完成 | 库存已更新 |
| 已取消 | cancelled | 事务取消 | 库存未更新 |
| 异常 | error | 处理异常 | 需要人工干预 |

#### 2.3.2 状态转换约束
1. **库存预留**：pending → processing → completed (库存预留)
2. **入库操作**：pending → processing → completed (库存增加)
3. **出库操作**：pending → processing → completed (库存减少)
4. **异常处理**：processing → error → pending/completed/cancelled

```python
class ReceivingInspection:
    def inspect_item(self, item, purchase_order_item):
        """收货检验逻辑"""
        
        inspection_result = {
            'passed': True,
            'failures': [],
            'actions': []
        }
        
        # 1. 数量检验
        if item['quantity'] != purchase_order_item['ordered_quantity']:
            if item['quantity'] < purchase_order_item['ordered_quantity'] * 0.95:
                inspection_result['passed'] = False
                inspection_result['failures'].append('数量严重不足')
                inspection_result['actions'].append('拒绝收货，通知采购部')
            else:
                inspection_result['actions'].append(f"数量差异: {item['quantity']}/{purchase_order_item['ordered_quantity']}")
        
        # 2. 质量检验
        if item.get('quality_status') == 'failed':
            inspection_result['passed'] = False
            inspection_result['failures'].append('质量不合格')
            inspection_result['actions'].append('隔离待处理')
        
        # 3. 包装检验
        if item.get('packaging_damaged'):
            inspection_result['actions'].append('包装损坏，记录照片')
        
        # 4. 效期检验
        if item.get('expiry_date'):
            days_remaining = (item['expiry_date'] - date.today()).days
            if days_remaining < 90:  # 效期不足90天
                inspection_result['actions'].append(f"近效期: {days_remaining}天")
                if days_remaining < 30:
                    inspection_result['failures'].append('效期严重不足')
                    inspection_result['passed'] = False
        
        return inspection_result
```

```python
class InventoryAllocationStrategy:
    def allocate_stock(self, spare_part_id, quantity, warehouse_id=None):
        """库存分配策略（先进先出+批次管理）"""
        
        # 获取可用库存批次
        available_batches = self.get_available_batches(
            spare_part_id, 
            warehouse_id
        )
        
        allocation_plan = []
        remaining_quantity = quantity
        
        # 策略1: 先进先出（按生产日期排序）
        for batch in sorted(available_batches, key=lambda x: x['manufacture_date']):
            if remaining_quantity <= 0:
                break
                
            batch_available = batch['available_quantity']
            allocate_quantity = min(batch_available, remaining_quantity)
            
            if allocate_quantity > 0:
                allocation_plan.append({
                    'batch_no': batch['batch_no'],
                    'warehouse_id': batch['warehouse_id'],
                    'location_id': batch['location_id'],
                    'quantity': allocate_quantity,
                    'unit_price': batch['unit_price'],
                    'expiry_date': batch['expiry_date']
                })
                remaining_quantity -= allocate_quantity
        
        # 策略2: 如果FIFO不足，使用最近批次
        if remaining_quantity > 0:
            for batch in sorted(available_batches, 
                                key=lambda x: x['manufacture_date'], 
                                reverse=True):
                if remaining_quantity <= 0:
                    break
                    
                batch_available = batch['available_quantity']
                # 跳过已分配的批次
                allocated = sum(p['quantity'] for p in allocation_plan 
                               if p['batch_no'] == batch['batch_no'])
                actual_available = batch_available - allocated
                
                if actual_available > 0:
                    allocate_quantity = min(actual_available, remaining_quantity)
                    allocation_plan.append({
                        'batch_no': batch['batch_no'],
                        'warehouse_id': batch['warehouse_id'],
                        'location_id': batch['location_id'],
                        'quantity': allocate_quantity,
                        'unit_price': batch['unit_price'],
                        'expiry_date': batch['expiry_date']
                    })
                    remaining_quantity -= allocate_quantity
        
        return {
            'allocation_plan': allocation_plan,
            'allocated_quantity': quantity - remaining_quantity,
            'shortage_quantity': remaining_quantity,
            'success': remaining_quantity == 0
        }
```

```python
class InventoryDiscrepancyAnalyzer:
    def analyze_discrepancy(self, system_qty, physical_qty, item_info):
        """盘点差异分析"""
        
        discrepancy = physical_qty - system_qty
        abs_discrepancy = abs(discrepancy)
        
        analysis = {
            'discrepancy': discrepancy,
            'abs_discrepancy': abs_discrepancy,
            'percentage': abs_discrepancy / system_qty * 100 if system_qty > 0 else 100,
            'severity': 'info',
            'possible_causes': [],
            'suggested_actions': []
        }
        
        # 差异严重程度判断
        if analysis['percentage'] > 50:
            analysis['severity'] = 'critical'
        elif analysis['percentage'] > 20:
            analysis['severity'] = 'high'
        elif analysis['percentage'] > 5:
            analysis['severity'] = 'medium'
        else:
            analysis['severity'] = 'low'
        
        # 可能原因分析
        if discrepancy > 0:  # 盘盈
            analysis['possible_causes'].extend([
                '入库未记账',
                '退货未处理',
                '盘点重复计数',
                '系统数据错误'
            ])
            analysis['suggested_actions'].append('调查原因后入库调整')
            
        else:  # 盘亏
            analysis['possible_causes'].extend([
                '出库未记账',
                '丢失或被盗',
                '报废未处理',
                '盘点漏计数',
                '系统数据错误'
            ])
            
            if item_info['is_controlled']:  # 受控件
                analysis['possible_causes'].append('序列号管理错误')
                analysis['suggested_actions'].append('核查序列号记录')
            
            if item_info['unit_value'] > 1000:  # 高价值物品
                analysis['severity'] = 'critical'
                analysis['suggested_actions'].append('启动安全调查')
            
            analysis['suggested_actions'].append('调查原因后出库调整')
        
        # 基于历史数据的分析
        history = self.get_discrepancy_history(item_info['spare_part_id'])
        if history:
            avg_discrepancy = sum(h['discrepancy'] for h in history) / len(history)
            if discrepancy * avg_discrepancy > 0:  # 同向差异
                analysis['possible_causes'].append('系统性误差')
                analysis['suggested_actions'].append('调整库存管理流程')
        
        return analysis
```

```python
class LowStockAlertRule:
    def check_low_stock(self, spare_part, warehouse_stock):
        """低库存预警检查"""
        
        alerts = []
        
        # 规则1: 低于安全库存
        if spare_part.safety_stock_level and warehouse_stock.available_stock < spare_part.safety_stock_level:
            alerts.append({
                'type': 'below_safety_stock',
                'severity': 'warning',
                'current': warehouse_stock.available_stock,
                'threshold': spare_part.safety_stock_level,
                'message': f"可用库存({warehouse_stock.available_stock})低于安全库存({spare_part.safety_stock_level})",
                'suggested_action': f"建议采购{spare_part.safety_stock_level * 2 - warehouse_stock.available_stock}件"
            })
        
        # 规则2: 低于最低库存
        if spare_part.min_stock_level and warehouse_stock.available_stock < spare_part.min_stock_level:
            alerts.append({
                'type': 'below_min_stock',
                'severity': 'error',
                'current': warehouse_stock.available_stock,
                'threshold': spare_part.min_stock_level,
                'message': f"可用库存({warehouse_stock.available_stock})低于最低库存({spare_part.min_stock_level})",
                'suggested_action': '立即采购'
            })
        
        # 规则3: 基于消耗速度的预警
        consumption_rate = self.calculate_consumption_rate(spare_part.id)
        days_of_supply = warehouse_stock.available_stock / consumption_rate if consumption_rate > 0 else 999
        
        if 0 < days_of_supply < spare_part.procurement_lead_time:
            alerts.append({
                'type': 'insufficient_lead_time',
                'severity': 'critical',
                'current_days': days_of_supply,
                'required_days': spare_part.procurement_lead_time,
                'message': f"当前库存仅能维持{days_of_supply:.1f}天，低于采购提前期({spare_part.procurement_lead_time}天)",
                'suggested_action': '紧急采购或调拨'
            })
        
        return alerts
```

```python
class EOQCalculator:
    def calculate_eoq(self, spare_part, demand_forecast):
        """计算经济订购量(EOQ)"""
        
        # EOQ公式: √(2DS/H)
        # D: 年需求量
        # S: 每次订购成本
        # H: 单位持有成本
        
        annual_demand = demand_forecast.get('annual_demand', spare_part.annual_consumption or 0)
        if annual_demand <= 0:
            return None
        
        ordering_cost = spare_part.ordering_cost or 100  # 默认订购成本100元
        holding_cost_rate = spare_part.holding_cost_rate or 0.2  # 默认持有成本率20%
        unit_cost = spare_part.standard_cost or spare_part.last_purchase_price or 0
        
        if unit_cost <= 0:
            return None
        
        holding_cost_per_unit = unit_cost * holding_cost_rate
        
        # 计算EOQ
        eoq = math.sqrt((2 * annual_demand * ordering_cost) / holding_cost_per_unit)
        
        # 考虑最小订购量和包装规格
        min_order_qty = spare_part.min_order_quantity or 1
        pack_size = spare_part.pack_size or 1
        
        # 调整EOQ
        adjusted_eoq = max(min_order_qty, eoq)
        adjusted_eoq = math.ceil(adjusted_eoq / pack_size) * pack_size
        
        return {
            'raw_eoq': eoq,
            'adjusted_eoq': adjusted_eoq,
            'annual_demand': annual_demand,
            'ordering_cost': ordering_cost,
            'holding_cost_per_unit': holding_cost_per_unit,
            'min_order_qty': min_order_qty,
            'pack_size': pack_size,
            'order_frequency': annual_demand / adjusted_eoq,  # 年订购次数
            'order_interval': 365 / (annual_demand / adjusted_eoq)  # 订购间隔(天)
        }
```

```python
class ReorderPointCalculator:
    def calculate_reorder_point(self, spare_part, lead_time_demand):
        """计算再订购点(ROP)"""
        
        # ROP = 提前期内的需求 + 安全库存
        
        lead_time = spare_part.procurement_lead_time or 30  # 天
        daily_demand = lead_time_demand.get('daily_demand', 0)
        
        if daily_demand <= 0:
            return None
        
        # 提前期内的需求
        lead_time_demand_qty = daily_demand * lead_time
        
        # 安全库存计算（基于服务水平）
        service_level = spare_part.service_level or 0.95  # 95%服务水平
        demand_std = lead_time_demand.get('demand_std', daily_demand * 0.3)  # 需求标准差
        
        # Z值（正态分布）
        z_value = self.get_z_value(service_level)
        
        # 安全库存 = Z * σ * √L
        safety_stock = z_value * demand_std * math.sqrt(lead_time)
        
        # 再订购点
        reorder_point = lead_time_demand_qty + safety_stock
        
        return {
```

#### 5.1.1 先进先出法 (FIFO)
```python
class FIFOCosting:
    def calculate_cost_of_goods_sold(self, issue_quantity, inventory_lots):
        """先进先出法计算出库成本"""
        
        cogs = 0  # 销售成本
        remaining_qty = issue_quantity
        consumed_lots = []
        
        # 按入库时间排序（先进先出）
        sorted_lots = sorted(inventory_lots, key=lambda x: x['received_date'])
        
        for lot in sorted_lots:
            if remaining_qty <= 0:
                break
            
            lot_available = lot['available_quantity']
            consume_qty = min(lot_available, remaining_qty)
            
            if consume_qty > 0:
                lot_cost = consume_qty * lot['unit_price']
                cogs += lot_cost
                remaining_qty -= consume_qty
                
                consumed_lots.append({
                    'lot_id': lot['id'],
                    'batch_no': lot['batch_no'],
                    'quantity': consume_qty,
                    'unit_price': lot['unit_price'],
                    'cost': lot_cost
                })
        
        return {
            'total_cost': cogs,
            'consumed_lots': consumed_lots,
            'remaining_qty': remaining_qty,
            'method': 'FIFO'
        }
```

#### 5.1.2 加权平均法 (Weighted Average)
```python
class WeightedAverageCosting:
    def calculate_weighted_average_cost(self, inventory_lots):
        """计算加权平均成本"""
        
        total_value = 0
        total_quantity = 0
        
        for lot in inventory_lots:
            total_value += lot['available_quantity'] * lot['unit_price']
            total_quantity += lot['available_quantity']
        
        if total_quantity > 0:
            weighted_avg_cost = total_value / total_quantity
        else:
            weighted_avg_cost = 0
        
        return {
            'weighted_avg_cost': weighted_avg_cost,
            'total_value': total_value,
            'total_quantity': total_quantity
        }
    
    def calculate_cost_of_goods_sold(self, issue_quantity, weighted_avg_cost):
        """加权平均法计算出库成本"""
        
        cogs = issue_quantity * weighted_avg_cost
        
        return {
            'total_cost': cogs,
            'unit_cost': weighted_avg_cost,
            'quantity': issue_quantity,
            'method': 'Weighted Average'
        }
```

#### 5.2.1 库存价值计算
```python
class InventoryValuation:
    def calculate_inventory_value(self, inventory_items, valuation_method='FIFO'):
        """计算库存价值"""
        
        total_value = 0
        valuation_details = []
        
        if valuation_method == 'FIFO':
            # 按批次计算
            for item in inventory_items:
                item_value = item['quantity'] * item['unit_price']
                total_value += item_value
                
                valuation_details.append({
                    'spare_part_id': item['spare_part_id'],
                    'batch_no': item['batch_no'],
                    'quantity': item['quantity'],
                    'unit_price': item['unit_price'],
                    'value': item_value,
                    'valuation_method': 'FIFO'
                })
        
        elif valuation_method == 'Weighted Average':
            # 按备件计算加权平均
            spare_part_groups = {}
            for item in inventory_items:
                spare_part_id = item['spare_part_id']
                if spare_part_id not in spare_part_groups:
                    spare_part_groups[spare_part_id] = []
                spare_part_groups[spare_part_id].append(item)
            
            for spare_part_id, items in spare_part_groups.items():
                # 计算加权平均成本
                total_qty = sum(item['quantity'] for item in items)
                total_cost = sum(item['quantity'] * item['unit_price'] for item in items)
                weighted_avg_cost = total_cost / total_qty if total_qty > 0 else 0
                
                spare_part_value = total_qty * weighted_avg_cost
                total_value += spare_part_value
                
                valuation_details.append({
                    'spare_part_id': spare_part_id,
                    'quantity': total_qty,
                    'weighted_avg_cost': weighted_avg_cost,
                    'value': spare_part_value,
                    'valuation_method': 'Weighted Average'
                })
        
        elif valuation_method == 'Lower of Cost or Market':
            # 成本与市价孰低法
            for item in inventory_items:
                cost_value = item['quantity'] * item['unit_price']
                market_price = self.get_market_price(item['spare_part_id'])
                market_value = item['quantity'] * market_price if market_price else cost_value
                
                item_value = min(cost_value, market_value)
                total_value += item_value
                
                valuation_details.append({
                    'spare_part_id': item['spare_part_id'],
                    'quantity': item['quantity'],
                    'cost_value': cost_value,
                    'market_value': market_value,
                    'value': item_value,
                    'valuation_method': 'LCM'
                })
        
        return {
            'total_value': total_value,
            'valuation_method': valuation_method,
            'details': valuation_details
        }
```

```python
class OfflineSyncManager:
    def sync_offline_data(self, device_id, operations):
        """同步离线操作数据"""
        
        sync_results = {
            'success': [],
            'failed': [],
            'conflicts': []
        }
        
        for op in operations:
            try:
                # 检查冲突
                conflict = self.check_conflict(op)
                if conflict:
                    sync_results['conflicts'].append({
                        'operation': op,
                        'conflict': conflict
                    })
                    continue
                
                # 执行业务操作
                result = self.execute_operation(op)
                
                if result['success']:
                    sync_results['success'].append({
                        'operation_id': op['local_id'],
                        'server_id': result['server_id'],
                        'timestamp': result['timestamp']
                    })
                else:
                    sync_results['failed'].append({
                        'operation_id': op['local_id'],
                        'error': result['error']
                    })
                    
            except Exception as e:
                sync_results['failed'].append({
                    'operation_id': op.get('local_id'),
                    'error': str(e)
                })
        
        return sync_results
    
    def check_conflict(self, operation):
        """检查操作冲突"""
        
        # 基于时间戳的冲突检测
        last_server_time = self.get_last_server_time(
            operation['entity_type'],
            operation['entity_id']
        )
        
        if operation['local_timestamp'] < last_server_time:
            return {
                'type': 'time_conflict',
                'local_time': operation['local_timestamp'],
                'server_time': last_server_time
            }
        
        # 基于版本的冲突检测
        if operation.get('version'):
            server_version = self.get_server_version(
                operation['entity_type'],
                operation['entity_id']
            )
            
            if operation['version'] != server_version:
                return {
                    'type': 'version_conflict',
                    'local_version': operation['version'],
                    'server_version': server_version
                }
        
        return None
```
