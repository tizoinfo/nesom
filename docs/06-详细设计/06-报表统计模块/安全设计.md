## 1. 安全设计概述

### 1.1 安全目标
报表统计模块处理NESOM系统的核心业务数据，包含敏感的经营数据和设备信息，必须满足以下安全目标：

1. **数据保密性**: 防止未授权访问报表数据，敏感数据脱敏显示
2. **数据完整性**: 确保统计计算结果准确，防止数据篡改
3. **访问控制**: 细粒度权限控制，行级数据权限过滤
4. **操作审计**: 所有报表查询、导出操作可追溯、可审计
5. **合规性**: 符合等保2.0三级、网络安全法、数据安全法要求
6. **隐私保护**: 个人数据最小化收集，敏感数据处理符合GDPR原则

### 1.2 安全威胁模型

#### 1.2.1 威胁识别
| 威胁类型 | 威胁描述 | 影响等级 | 可能性 | 应对措施 |
|----------|----------|----------|--------|----------|
| SQL注入 | 通过报表参数注入恶意SQL | 严重 | 中 | 参数化查询，查询白名单 |
| 数据泄露 | 未授权访问敏感报表数据 | 严重 | 中 | 行级权限，数据脱敏 |
| 拒绝服务 | 复杂查询消耗系统资源 | 高 | 高 | 查询限制，资源配额 |
| 权限提升 | 绕过权限访问高敏感数据 | 严重 | 低 | 权限验证，操作审计 |
| 导出滥用 | 批量导出敏感数据 | 高 | 中 | 导出审批，数量限制 |
| 缓存投毒 | 污染缓存返回错误数据 | 中 | 低 | 缓存验证，数据签名 |
| 内部威胁 | 内部人员滥用报表权限 | 高 | 中 | 操作审计，权限分离 |

#### 1.2.2 攻击面分析
1. **报表查询接口**: SQL注入，参数篡改，权限绕过
2. **数据导出功能**: 数据泄露，导出滥用，格式解析漏洞
3. **异步任务系统**: 任务注入，资源耗尽，结果篡改
4. **缓存系统**: 缓存投毒，缓存穿透，缓存击穿
5. **文件存储**: 文件上传漏洞，路径遍历，文件包含
6. **WebSocket连接**: 未授权订阅，消息注入，连接劫持
7. **第三方集成**: 图表库漏洞，导出库漏洞，地图服务

### 1.3 安全设计原则
1. **最小权限原则**: 报表用户仅能访问必要的场站和数据
2. **纵深防御**: 前端验证 + 后端验证 + 数据库权限 + 审计日志
3. **默认拒绝**: 默认不允许访问，显式授权才允许
4. **数据最小化**: 只收集和处理必要的业务数据
5. **隐私设计**: 隐私保护融入系统设计，非事后添加
6. **安全左移**: 安全需求在需求分析阶段就明确
7. **零信任**: 不信任任何查询请求，每次都需要验证

```python
def apply_row_level_security(base_query: str, user_permissions: DataPermission) -> str:
    """
    在查询中自动注入行级权限过滤条件
    """
    filters = []
    
    # 场站权限过滤
    if user_permissions.data_scope.stations:
        station_ids = ','.join([f"'{sid}'" for sid in user_permissions.data_scope.stations])
        filters.append(f"station_id IN ({station_ids})")
    elif user_permissions.data_scope.station_groups:
        # 基于场站组的过滤
        group_condition = build_station_group_condition(user_permissions.data_scope.station_groups)
        filters.append(group_condition)
    
    # 设备类型权限过滤
    if user_permissions.data_scope.device_types:
        device_type_ids = ','.join([f"'{tid}'" for tid in user_permissions.data_scope.device_types])
        filters.append(f"device_type_id IN ({device_type_ids})")
    
    # 时间范围权限
    if not user_permissions.data_scope.time_range.allow_historical:
        max_days = user_permissions.data_scope.time_range.max_days
        cutoff_date = f"CURRENT_DATE - INTERVAL {max_days} DAY"
        filters.append(f"data_date >= {cutoff_date}")
    
    # 组合过滤条件
    if filters:
        where_clause = " AND ".join(filters)
        if "WHERE" in base_query.upper():
            # 已有WHERE子句，添加AND条件
            base_query = base_query.replace("WHERE", f"WHERE {where_clause} AND", 1)
        else:
            # 添加WHERE子句
            base_query = f"{base_query} WHERE {where_clause}"
    
    return base_query
```

```python
def mask_sensitive_data(data: List[Dict], user_permissions: DataPermission) -> List[Dict]:
    """
    根据用户权限脱敏敏感数据
    """
    masked_data = []

        masked_row = row.copy()

                masked_row['unit_cost'] = '***'

                masked_row['labor_cost'] = '***'
        
        # 收益数据脱敏
        if not user_permissions.data_scope.dimensions.revenue:
            if 'revenue' in masked_row:
                masked_row['revenue'] = '***'
            if 'profit' in masked_row:
                masked_row['profit'] = '***'
            if 'roi' in masked_row:
                masked_row['roi'] = '***'
```

```python
    def __init__(self):
        self.approval_rules = {
            'sensitive': {
                'cost_data': ['department_manager', 'finance_manager'],
                'revenue_data': ['department_manager', 'finance_manager', 'general_manager'],
                'employee_data': ['hr_manager', 'department_manager'],

            },
            'frequency': {
                'first_time': ['department_manager'],
                'repeated': ['data_governance']  # 频繁导出需要额外审批
            }
        }
    
    def requires_approval(self, export_request: ExportRequest) -> bool:
        """
        判断导出请求是否需要审批
        """
        # 敏感数据类型检查
        if export_request.data_type in self.approval_rules['sensitive']:
            return True
        
        # 数据量检查
        if export_request.estimated_rows > 10000:
            return True
        
        # 导出频率检查
        recent_exports = self.get_recent_exports(export_request.user_id, hours=24)
        if len(recent_exports) >= 5:  # 24小时内超过5次导出
            return True
        
        return False
    
    def initiate_approval(self, export_request: ExportRequest) -> str:
        """
        发起审批流程
        """
        # 确定审批人
        approvers = self.determine_approvers(export_request)
        
        # 创建审批任务
        approval_task = ApprovalTask(
            request_id=export_request.id,
            requestor_id=export_request.user_id,
            approvers=approvers,
            data_type=export_request.data_type,
            estimated_rows=export_request.estimated_rows,
            expiry_time=datetime.utcnow() + timedelta(hours=24)
        )
        
        # 保存到数据库
        db.session.add(approval_task)
        db.session.commit()
        
        # 发送审批通知
        self.send_approval_notifications(approvers, export_request)
```

### 3.1 SQL注入防护

#### 3.1.1 参数化查询强制实施
所有动态查询必须使用参数化查询：

```python
def execute_safe_query(template: ReportTemplate, parameters: Dict) -> List[Dict]:
    """
    安全的参数化查询执行
    """
    # 1. 获取查询模板
    sql_template = template.data_source_config.query
    
    # 2. 参数验证和类型转换
    validated_params = validate_parameters(template.parameter_definitions, parameters)
    
    # 3. 构建参数化查询
    # 使用SQLAlchemy的text()和bindparams
    from sqlalchemy import text
    
    # 将命名参数转换为位置参数格式
    param_bindings = {}
    for param_name, param_value in validated_params.items():
        # 参数类型安全转换
        if isinstance(param_value, str):
            # 字符串参数：防止注入
            param_bindings[param_name] = str(param_value)
        elif isinstance(param_value, (int, float)):
            param_bindings[param_name] = param_value
        elif isinstance(param_value, date):
            param_bindings[param_name] = param_value.isoformat()
        else:
            raise ValueError(f"Unsupported parameter type: {type(param_value)}")
    
    # 4. 执行参数化查询
    safe_sql = text(sql_template)
    result = db.session.execute(safe_sql, param_bindings)
    
    # 5. 转换为字典列表
    columns = result.keys()
    rows = []
    for row in result:
        rows.append(dict(zip(columns, row)))
    
    return rows
```

#### 3.1.2 查询模板白名单
仅允许预定义的查询模板：

```python
class QueryTemplateWhitelist:
    """
    查询模板白名单管理
    """
    
    def __init__(self):
        self.allowed_templates = self.load_whitelist()
    
    def load_whitelist(self) -> Set[str]:
        """
        加载预定义的查询模板
        """
        # 从数据库加载已审核的查询模板
        templates = ReportTemplate.query.filter(
            ReportTemplate.is_active == True,
            ReportTemplate.approval_status == 'approved'
        ).all()
        
        # 生成模板哈希指纹
        template_hashes = set()
        for template in templates:
            # 基于模板内容和数据源生成唯一哈希
            template_fingerprint = self.generate_fingerprint(
                template.data_source_config.query,
                template.data_source_type
            )
            template_hashes.add(template_fingerprint)
        
        return template_hashes
    
    def is_template_allowed(self, sql_template: str, data_source_type: str) -> bool:
        """
        检查查询模板是否在白名单中
        """
        fingerprint = self.generate_fingerprint(sql_template, data_source_type)
        return fingerprint in self.allowed_templates
    
    def generate_fingerprint(self, sql: str, data_source_type: str) -> str:
        """
        生成查询模板指纹
        """
        # 标准化SQL（去除注释、空白字符、统一大小写）
        normalized_sql = self.normalize_sql(sql)
        
        # 添加数据源类型
        content = f"{data_source_type}:{normalized_sql}"
        
        # 生成SHA256哈希
        import hashlib
        return hashlib.sha256(content.encode('utf-8')).hexdigest()
    
    def normalize_sql(self, sql: str) -> str:
        """
        标准化SQL语句
        """
        # 移除SQL注释
        import re
        
        # 移除单行注释
        sql = re.sub(r'--.*$', '', sql, flags=re.MULTILINE)
        
        # 移除多行注释
        sql = re.sub(r'/\*.*?\*/', '', sql, flags=re.DOTALL)
        
        # 标准化空白字符
        sql = re.sub(r'\s+', ' ', sql).strip()
        
        # 统一关键字为大写（可选）
        keywords = ['SELECT', 'FROM', 'WHERE', 'JOIN', 'GROUP BY', 'ORDER BY', 'LIMIT']
        for keyword in keywords:
            sql = re.sub(rf'\b{keyword}\b', keyword, sql, flags=re.IGNORECASE)
        
        return sql
```

```python
            },
            'admin': {
                'max_execution_time': 600,
                'max_result_rows': 1000000,
                'max_memory_mb': 2000,
                'max_columns': 200,
                'daily_query_count': 5000
            }
        }
    
    def enforce_limits(self, user: User, query_plan: Dict) -> bool:
        """
        强制执行查询限制
        """
        user_limits = self.resource_limits.get(user.role, self.resource_limits['viewer'])
        
        # 1. 执行时间限制
        if query_plan.get('estimated_time', 0) > user_limits['max_execution_time']:
            raise QueryLimitExceeded(f"预计执行时间超过限制: {user_limits['max_execution_time']}秒")
        
        # 2. 结果行数限制
        if query_plan.get('estimated_rows', 0) > user_limits['max_result_rows']:
            # 强制添加LIMIT子句
            self.force_limit_clause(query_plan, user_limits['max_result_rows'])
        
        # 3. 内存使用限制
        if query_plan.get('estimated_memory_mb', 0) > user_limits['max_memory_mb']:
            raise QueryLimitExceeded(f"预计内存使用超过限制: {user_limits['max_memory_mb']}MB")
        
        # 4. 列数限制
        if query_plan.get('column_count', 0) > user_limits['max_columns']:
            raise QueryLimitExceeded(f"返回列数超过限制: {user_limits['max_columns']}")
        
        # 5. 每日查询次数检查
        daily_count = self.get_daily_query_count(user.id)
        if daily_count >= user_limits['daily_query_count']:
            raise QueryLimitExceeded(f"每日查询次数超过限制: {user_limits['daily_query_count']}")
        
        return True
```

#### 3.2.2 查询超时控制
```python
class QueryTimeoutController:
    """
    查询超时控制
    """
    
    def execute_with_timeout(self, query_func, timeout_seconds: int, *args, **kwargs):
        """
        带超时控制的查询执行
        """
        import signal
        import threading
        
        class TimeoutException(Exception):
            pass
        
        def timeout_handler(signum, frame):
            raise TimeoutException(f"Query timeout after {timeout_seconds} seconds")
        
        # 设置超时信号
        signal.signal(signal.SIGALRM, timeout_handler)
        signal.alarm(timeout_seconds)
        
        try:
            # 在子线程中执行查询
            result = None
            exception = None
            
            def query_runner():
                nonlocal result, exception
                try:
                    result = query_func(*args, **kwargs)
                except Exception as e:
                    exception = e
            
            thread = threading.Thread(target=query_runner)
            thread.start()
            thread.join(timeout_seconds)
            
            if thread.is_alive():
                # 查询超时，终止数据库连接
                self.terminate_query_connection()
                raise TimeoutException(f"Query timeout after {timeout_seconds} seconds")
            
            if exception:
                raise exception
            
            return result
            
        finally:
            # 取消超时信号
            signal.alarm(0)
```

### 5.1 缓存数据验证

```python
class CacheSecurityValidator:
    """
    缓存数据安全验证
    """
    
    def validate_cache_data(self, cache_key: str, cached_data: Any, user: User) -> bool:
        """
        验证缓存数据的安全性和有效性
        """
        # 1. 检查缓存键是否被篡改
        if not self.validate_cache_key_integrity(cache_key):
            return False
        
        # 2. 检查缓存数据签名
        if not self.validate_data_signature(cached_data):
            return False
        
        # 3. 检查缓存数据权限
        if not self.validate_data_permissions(cached_data, user):
            return False
        
        # 4. 检查缓存数据时效性
        if not self.validate_data_freshness(cached_data):
            return False
        
        # 5. 检查缓存数据完整性
        if not self.validate_data_integrity(cached_data):
            return False
        
        return True
    
    def validate_cache_key_integrity(self, cache_key: str) -> bool:
        """
        验证缓存键完整性（防止缓存键篡改）
        """
        # 缓存键格式：type:hash:signature
        parts = cache_key.split(':')
        if len(parts) != 3:
            return False
        
        cache_type, data_hash, signature = parts
        
        # 验证签名
        expected_signature = self.sign_cache_key(cache_type, data_hash)
        return signature == expected_signature
    
    def sign_cache_key(self, cache_type: str, data_hash: str) -> str:
        """
        为缓存键生成签名
        """
        import hmac
        import hashlib
        
        secret_key = self.get_cache_secret_key()
        message = f"{cache_type}:{data_hash}"
        
        signature = hmac.new(
            secret_key.encode('utf-8'),
            message.encode('utf-8'),
            hashlib.sha256
        ).hexdigest()
        
        # 返回前8位作为签名
        return signature[:8]
```

### 5.2 缓存穿透防护
```python
class CachePenetrationProtector:
    """
    缓存穿透防护
    """
    
    def __init__(self):
        # 布隆过滤器（判断键是否可能存在）
        self.bloom_filter = BloomFilter(capacity=1000000, error_rate=0.001)
        
        # 空结果缓存（防止反复查询不存在的数据）
        self.null_cache = {}
        self.null_cache_ttl = 300  # 5分钟
    
    def protect(self, cache_key: str, query_func, *args, **kwargs):
        """
        防止缓存穿透的查询包装器
        """
        # 1. 检查布隆过滤器
        if not self.bloom_filter.exists(cache_key):
            # 键肯定不存在，直接返回空结果
            return self.get_null_result()
        
        # 2. 检查空结果缓存
        if cache_key in self.null_cache:
            null_entry = self.null_cache[cache_key]
            if time.time() - null_entry['timestamp'] < self.null_cache_ttl:
                # 返回缓存的空结果
                return null_entry['result']
            else:
                # 空结果缓存过期，移除
                del self.null_cache[cache_key]
        
        # 3. 执行实际查询
        try:
            result = query_func(*args, **kwargs)
            
            if not result:
                # 空结果，记录到空结果缓存
                self.null_cache[cache_key] = {
                    'result': self.get_null_result(),
                    'timestamp': time.time()
                }
                # 从布隆过滤器中移除（可能误判）
                self.bloom_filter.remove(cache_key)
            else:
                # 有结果，添加到布隆过滤器
                self.bloom_filter.add(cache_key)
            
            return result
            
        except Exception as e:
            # 查询异常，记录到布隆过滤器（避免反复查询）
            self.bloom_filter.remove(cache_key)
            raise e
    
    def get_null_result(self):
        """
        获取标准的空结果
        """
        return {
            'data': [],
            'summary': {
                'total_rows': 0,
                'cache_hit': False,
                'null_result': True
            }
        }
```

```python
class SecurityAnomalyDetector:
    """
    安全异常检测

    def __init__(self):

                'name': 'unusual_time_access',
                'metric': 'access_outside_hours',
                'threshold': 1,  # 非工作时间访问
                'window': '1d',
                'action': 'alert'
            },
            
            # 规则4: 权限提升尝试
            {
                'name': 'privilege_escalation_attempt',
                'metric': 'unauthorized_template_access',

                'window': '10m',
                'action': 'block_and_alert'
            },
            
            # 规则5: SQL注入尝试
            {
                'name': 'sql_injection_attempt',
                'metric': 'suspicious_parameters',
                'threshold': 1,  # 1次尝试
                'window': '1m',
                'action': 'block_and_report'
            }
        ]
    
    def detect_anomalies(self, audit_logs: List[AuditLog]) -> List[Anomaly]:
        """
        检测安全异常
        """
        anomalies = []
        
        for rule in self.detection_rules:
            # 应用检测规则
            if self.check_rule_violation(rule, audit_logs):
                anomaly = Anomaly(
                    rule_name=rule['name'],
                    severity=self.calculate_severity(rule),
                    detected_at=datetime.utcnow(),
                    evidence=audit_logs,
                    recommended_action=rule['action']
                )
                anomalies.append(anomaly)
                
                # 触发响应动作
                self.trigger_response(rule['action'], anomaly)
        
        return anomalies
    
    def trigger_response(self, action: str, anomaly: Anomaly):
        """
        触发安全响应
        """
        if action == 'alert':
            self.send_alert(anomaly)
        elif action == 'block_and_alert':
            self.block_user(anomaly.user_id)
            self.send_alert(anomaly)
            self.create_incident_report(anomaly)
        elif action == 'block_and_report':
            self.block_user(anomaly.user_id)
            self.send_alert(anomaly)
            self.create_incident_report(anomaly)
            self.notify_security_team(anomaly)
```
