## 1. 设计概述

### 1.1 设计目标
报表统计前端模块提供强大、灵活、美观的数据分析和可视化界面，支持：
- **报表构建**: 拖拽式报表设计，类似Power BI的体验
- **数据可视化**: 丰富的图表类型，实时数据刷新
- **仪表盘定制**: 个性化仪表盘布局和主题
- **统计分析**: 多维度数据钻取和分析
- **移动端适配**: 响应式设计，支持移动端查看和简单编辑

### 1.2 设计原则
1. **用户体验优先**: 直观的拖拽操作，实时预览，即时反馈
2. **可视化表达**: 数据驱动设计，选择合适的图表表达数据
3. **性能优化**: 大数据量虚拟滚动，图表懒加载，渐进渲染
4. **灵活性**: 支持自定义图表、主题、布局
5. **一致性**: 统一的交互模式，视觉风格，组件规范
6. **可访问性**: 支持键盘操作，屏幕阅读器，高对比度主题

### 1.3 技术约束
- **框架**: Vue 3.4 Composition API + `<script setup>`
- **语言**: TypeScript 5.3 (严格模式)
- **UI库**: Element Plus 2.7 + 自定义主题
- **状态管理**: Pinia (模块化状态管理)
- **路由**: Vue Router 4
- **HTTP客户端**: Axios + 自定义封装
- **图表库**: ECharts 5.4 + Vue-ECharts 6.6
- **拖拽库**: Vue.Draggable 4.1 (基于Sortable.js)
- **网格布局**: Vue Grid Layout 3.0
- **表格组件**: vxe-table 4.7 (支持虚拟滚动)
- **导出功能**: SheetJS (Excel), jsPDF (PDF)
- **WebSocket**: 原生WebSocket + 自动重连 (实时数据)

#### 3.5.2 计划类型

2. **自定义Cron表达式**:
   - 支持标准Cron语法

3. **事件触发**:
   - 数据更新时触发
   - 阈值达到时触发

### 4.1 虚拟滚动大数据表格
**场景**: 表格数据超过10万行

```typescript

<template>

    ref="tableRef"
    :data="virtualData"
    :height="tableHeight"

import { ref, computed, onMounted } from 'vue'

  columns: TableColumn[]
  pageSize?: number
}>()

const tableRef = ref<VxeTableInstance>()
const scrollTop = ref(0)
const visibleStartIndex = ref(0)

// 虚拟数据计算
const virtualData = computed(() => {
  const start = Math.max(0, visibleStartIndex.value - 10) // 预加载10行
  const end = Math.min(props.data.length, visibleStartIndex.value + props.pageSize! + 10)
  return props.data.slice(start, end)

}
```

### 4.2 图表懒加载和渐进渲染
**策略**:

2. **分步渲染**: 大数据量图表先渲染概要，再渐进细化
3. **Web Worker**: 复杂计算在Worker线程进行

```typescript
// LazyChart.vue
<script setup lang="ts">
import { ref, onMounted, onUnmounted } from 'vue'

const props = defineProps<{
  chartId: string
  data: ChartData
  threshold?: number
}>()

const chartRef = ref<HTMLElement>()
const isVisible = ref(false)
let observer: IntersectionObserver

onMounted(() => {
  // 创建Intersection Observer
  observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        isVisible.value = true
        observer.unobserve(entry.target) // 触发后停止观察
      }
    })
  }, {
    threshold: props.threshold || 0.1,
    rootMargin: '50px' // 提前50px加载
  })
  
  if (chartRef.value) {
    observer.observe(chartRef.value)
  }
})

onUnmounted(() => {
  if (observer) {
    observer.disconnect()
  }
})
</script>
```

### 4.3 数据缓存和本地存储
**缓存策略**:
1. **内存缓存**: Vue reactive对象存储热点数据
2. **LocalStorage**: 用户偏好设置、最近查询
3. **IndexedDB**: 大数据集、历史记录
4. **Service Worker**: 离线缓存，PWA支持

```typescript
// DataCache.ts
class DataCache {
  private memoryCache = new Map<string, { data: any; timestamp: number }>()
  private memoryTTL = 5 * 60 * 1000 // 5分钟
  
  // 内存缓存
  setMemory(key: string, data: any): void {
    this.memoryCache.set(key, {
      data,
      timestamp: Date.now()
    })
  }
  
  getMemory(key: string): any {
    const item = this.memoryCache.get(key)
    if (!item) return null
    
    // 检查是否过期
    if (Date.now() - item.timestamp > this.memoryTTL) {
      this.memoryCache.delete(key)
      return null
    }
    
    return item.data
  }
  
  // LocalStorage缓存
  setLocal(key: string, data: any): void {
    try {
      localStorage.setItem(`report_cache_${key}`, JSON.stringify({
        data,
        timestamp: Date.now()
      }))
    } catch (e) {
      console.warn('LocalStorage存储失败:', e)
    }
  }
  
  getLocal(key: string): any {
    try {
      const item = localStorage.getItem(`report_cache_${key}`)
      if (!item) return null
      
      const parsed = JSON.parse(item)
      const localTTL = 24 * 60 * 60 * 1000 // 24小时
      
      if (Date.now() - parsed.timestamp > localTTL) {
        localStorage.removeItem(`report_cache_${key}`)
        return null
      }
      
      return parsed.data
    } catch (e) {
      console.warn('LocalStorage读取失败:', e)
      return null
    }
  }
}

export const dataCache = new DataCache()
```

```typescript
// ExportManager.ts
class ExportManager {
  async exportToExcel(data: any[], options: ExportOptions): Promise<Blob> {
    const wb = XLSX.utils.book_new()
    
    // 数据转换
    const ws = XLSX.utils.json_to_sheet(data)
    
    // 添加样式
    if (options.styles) {
      this.applyExcelStyles(ws, options.styles)
    }
    
    // 添加图表
    if (options.charts) {
      await this.addChartsToExcel(wb, ws, options.charts)
    }
    
    XLSX.utils.book_append_sheet(wb, ws, options.sheetName || 'Sheet1')
    
    // 生成文件
    const buffer = XLSX.write(wb, { type: 'array', bookType: 'xlsx' })
    return new Blob([buffer], { type: 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet' })
  }
  
  async exportToPDF(element: HTMLElement, options: PdfOptions): Promise<Blob> {
    const { jsPDF } = await import('jspdf')
    const doc = new jsPDF({
      orientation: options.landscape ? 'landscape' : 'portrait',
      unit: 'mm',
      format: options.format || 'a4'
    })
    
    // 添加水印
    if (options.watermark) {
      this.addWatermark(doc, options.watermark)
    }
    
    // 使用html2canvas转换HTML为图片
    const canvas = await html2canvas(element, {
      scale: 2,
      useCORS: true,
      logging: false
    })
    
    const imgData = canvas.toDataURL('image/png')
    const imgProps = doc.getImageProperties(imgData)
    const pdfWidth = doc.internal.pageSize.getWidth()
    const pdfHeight = (imgProps.height * pdfWidth) / imgProps.width
    
    doc.addImage(imgData, 'PNG', 0, 0, pdfWidth, pdfHeight)
    
    return doc.output('blob')
  }
}
```
