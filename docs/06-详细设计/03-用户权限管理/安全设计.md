## 1. 安全设计概述

### 1.1 安全目标
用户权限管理模块是NESOM系统的安全核心，必须满足以下安全目标：

1. **身份可信**：确保用户身份真实可信，防止身份伪造和冒用
2. **权限可控**：实现细粒度权限控制，防止越权访问
3. **操作可溯**：所有安全相关操作完整记录，支持审计追溯
4. **数据保密**：保护用户敏感信息（密码、MFA密钥、个人数据）
5. **服务可用**：保障认证授权服务高可用，防止拒绝服务攻击
6. **合规遵从**：满足等保2.0三级、GDPR、个人信息保护法要求

### 1.2 安全威胁模型

#### 1.2.1 威胁识别
| 威胁类型 | 威胁描述 | 影响等级 | 可能性 | 防护措施 |
|----------|----------|----------|--------|----------|
| 凭证泄露 | 密码、令牌泄露导致账户被接管 | 严重 | 高 | 多因素认证、短期令牌、加密存储 |
| 暴力破解 | 自动化尝试破解用户密码 | 高 | 高 | 账户锁定、IP限制、验证码 |
| 会话劫持 | 窃取会话令牌进行未授权访问 | 严重 | 中 | JWT短期有效、Token黑名单、HTTPS |
| 权限提升 | 攻击者获取超过自身权限的访问能力 | 严重 | 中 | 最小权限原则、服务端权限验证 |
| 数据泄露 | 用户敏感信息（密码哈希、个人信息）泄露 | 严重 | 中 | 加密存储、数据脱敏、访问控制 |
| 拒绝服务 | 认证服务被攻击导致无法登录 | 高 | 高 | 限流、熔断、服务降级 |
| 内部威胁 | 内部人员滥用权限或泄露数据 | 高 | 中 | 职责分离、操作审计、权限复核 |
| 社会工程 | 通过欺骗手段获取凭证或权限 | 中 | 高 | 安全意识培训、二次验证 |

#### 1.2.2 攻击面分析
1. **认证接口**：登录、注册、密码重置、Token刷新
2. **用户管理**：用户增删改查、密码修改、账户锁定
3. **权限管理**：角色分配、权限配置、数据权限设置
4. **会话管理**：会话查看、强制登出、令牌撤销
5. **审计日志**：日志查询、数据导出、报告生成
6. **系统配置**：安全策略配置、密码策略修改

### 1.3 安全设计原则
1. **零信任原则**：不信任任何内部或外部实体，始终验证
2. **最小权限原则**：用户和进程仅拥有必要的最小权限
3. **纵深防御**：多层安全防护，单一防线失效不影响整体安全
4. **默认安全**：默认配置即为安全配置，需明确降低安全性
5. **安全左移**：安全考虑贯穿需求、设计、开发、测试、运维全生命周期
6. **隐私保护**：数据最小化收集，用户知情同意，数据可删除
7. **失效安全**：系统故障时进入安全状态（如拒绝访问）

class BruteForceProtection:
    def __init__(self):
        self.redis = get_redis_client()
        
    def check_login_attempt(self, username: str, ip_address: str) -> bool:
        """检查登录尝试，返回是否允许"""
        # 用户维度检查
        user_key = f"login:user:{username}"
        user_attempts = self.redis.get(user_key) or 0
        
        # IP维度检查  
        ip_key = f"login:ip:{ip_address}"
        ip_attempts = self.redis.get(ip_key) or 0
        
        # 动态阈值（根据历史行为调整）
        user_threshold = self.calculate_threshold(username, "user")
        ip_threshold = self.calculate_threshold(ip_address, "ip")
        
        if int(user_attempts) >= user_threshold:
            return False, "user_locked"
            
        if int(ip_attempts) >= ip_threshold:
            return False, "ip_locked"
            
        return True, "allowed"
    
    def record_failed_attempt(self, username: str, ip_address: str):
        """记录失败尝试"""
        # 用户维度计数（5分钟窗口）
        user_key = f"login:user:{username}"
        self.redis.incr(user_key)
        self.redis.expire(user_key, 300)  # 5分钟
        
        # IP维度计数（1小时窗口）
        ip_key = f"login:ip:{ip_address}"
        self.redis.incr(ip_key)
        self.redis.expire(ip_key, 3600)  # 1小时
        
        # 全局计数（防御分布式攻击）
        global_key = "login:global:failures"
        self.redis.incr(global_key)
        self.redis.expire(global_key, 60)  # 1分钟

### 2.3 密码安全实施

#### 2.3.1 密码哈希实现
```python
import bcrypt
import hashlib
import secrets

class PasswordHasher:
    def __init__(self, cost_factor: int = 12):
        self.cost_factor = cost_factor
        
    def hash_password(self, password: str) -> str:
        """使用bcrypt哈希密码"""
        # 密码规范化（防止编码问题）
        password_bytes = password.encode('utf-8')
        
        # 生成salt
        salt = bcrypt.gensalt(rounds=self.cost_factor)
        
        # 计算哈希
        hashed = bcrypt.hashpw(password_bytes, salt)
        
        # 返回字符串格式
        return hashed.decode('utf-8')
    
    def verify_password(self, password: str, hashed_password: str) -> bool:
        """验证密码"""
        try:
            password_bytes = password.encode('utf-8')
            hashed_bytes = hashed_password.encode('utf-8')
            return bcrypt.checkpw(password_bytes, hashed_bytes)
        except Exception:
            # 防止时序攻击，固定时间返回
            self._dummy_verify()
            return False
    
    def _dummy_verify(self):
        """虚拟验证，防止时序攻击"""
        bcrypt.hashpw(b"dummy", bcrypt.gensalt())
```

#### 2.3.2 密码策略执行
```python
class PasswordValidator:
    def __init__(self, policy_config: dict):
        self.policy = policy_config
        
    def validate(self, password: str, username: str = None, old_passwords: list = None) -> tuple[bool, list[str]]:
        """验证密码是否符合策略"""
        errors = []
        
        # 长度检查
        if len(password) < self.policy["min_length"]:
            errors.append(f"密码至少需要{self.policy['min_length']}位")
        
        # 复杂度检查
        if self.policy["require_uppercase"] and not any(c.isupper() for c in password):
            errors.append("必须包含大写字母")
            
        if self.policy["require_lowercase"] and not any(c.islower() for c in password):
            errors.append("必须包含小写字母")
            
        if self.policy["require_digits"] and not any(c.isdigit() for c in password):
            errors.append("必须包含数字")
            
        # 禁止包含用户名
        if self.policy["not_contain_username"] and username:
            if username.lower() in password.lower():
                errors.append("密码不能包含用户名")
        
        # 历史密码检查
        if self.policy["history_size"] > 0 and old_passwords:
            for old_hash in old_passwords[-self.policy["history_size"]:]:
                if bcrypt.checkpw(password.encode(), old_hash.encode()):
                    errors.append(f"不能使用最近{self.policy['history_size']}次用过的密码")
                    break
        
        # 常用密码检查
        if self.policy["not_contain_common_words"]:
            if self.is_common_password(password):
                errors.append("密码过于常见，请使用更复杂的密码")
        
        return len(errors) == 0, errors
```

class PermissionValidator:
    def __init__(self, cache_enabled: bool = True):
        self.cache = RedisCache() if cache_enabled else None
        self.permission_hierarchy = self.load_hierarchy()
        
    def check_permission(self, user_id: str, permission_code: str, resource_id: str = None) -> bool:
        """检查用户是否有指定权限"""
        # 1. 超级管理员直接通过
        if self.is_superadmin(user_id):
            return True
            
        # 2. 检查缓存
        cache_key = f"user:perms:{user_id}"
        if self.cache:
            cached_perms = self.cache.get(cache_key)
            if cached_perms:
                return permission_code in cached_perms
        
        # 3. 查询数据库获取用户权限
        user_perms = self.get_user_permissions_from_db(user_id)
        
        # 4. 检查权限（支持通配符和继承）
        has_perm = self.evaluate_permission(permission_code, user_perms)
        
        # 5. 更新缓存
        if self.cache and has_perm:
            self.cache.set(cache_key, user_perms, ttl=300)  # 5分钟
        
        return has_perm
    
    def evaluate_permission(self, required: str, user_perms: set) -> bool:
        """评估权限，支持通配符和层级继承"""
        # 直接匹配
        if required in user_perms:
            return True
            
        # 通配符匹配（如 device:* 匹配 device:read等）
        parts = required.split(':')
        for i in range(len(parts) - 1, 0, -1):
            wildcard = ':'.join(parts[:i]) + ':*'
            if wildcard in user_perms:
                return True
                
        # 权限继承检查（如 manage 包含 read, write）
        if required in self.permission_hierarchy.get_inherited():
            inherited = self.permission_hierarchy.get_inherits(required)
            return any(perm in user_perms for perm in inherited)
            
        return False
```

        "system:config:update": {"level": "critical", "2fa_required": True},
        
        # 数据操作
        "data:export": {"level": "medium", "2fa_required": False},
        "audit:purge": {"level": "high", "2fa_required": True},
    }
    
    def check_operation(self, user_id: str, operation: str, context: dict = None) -> dict:
        """检查敏感操作是否需要二次验证"""
        operation_config = self.OPERATIONS_REQUIRE_2FA.get(operation)
        
        if not operation_config:
            return {"require_2fa": False, "reason": None}
        
        # 检查用户风险等级
        risk_level = self.assess_user_risk(user_id, context)
        
        # 根据风险等级决定是否需要2FA
        require_2fa = operation_config["2fa_required"]
        
        # 高风险用户的所有敏感操作都需要2FA
        if risk_level == "high":
            require_2fa = True
            
        return {
            "require_2fa": require_2fa,
            "reason": operation_config.get("reason", "敏感操作需要二次验证"),
            "operation_level": operation_config["level"],
            "risk_level": risk_level
        }
```

### 4.1 JWT安全实施

#### 4.1.1 JWT生成和验证
```python
import jwt
import datetime
from cryptography.hazmat.primitives import serialization

class JWTSecurityService:
    def __init__(self):
        # 加载密钥（定期轮换）
        self.private_key = self.load_private_key()
        self.public_key = self.load_public_key()
        self.key_version = "v1_202604"
        
    def create_access_token(self, user: User, session_id: str) -> str:
        """创建访问令牌"""
        now = datetime.datetime.utcnow()
        
        payload = {
            # 标准声明
            "iss": "nesom-auth",  # 签发者
            "sub": user.id,       # 主题（用户ID）
            "aud": ["nesom-api"], # 受众
            "exp": now + datetime.timedelta(seconds=900),  # 15分钟过期
            "nbf": now,           # 不早于
            "iat": now,           # 签发时间
            
            # 自定义声明
            "jti": session_id,    # 令牌唯一标识
            "typ": "access",      # 令牌类型
            "ver": self.key_version,  # 密钥版本
            
            # 用户信息
            "username": user.username,
            "roles": user.roles,
            "perms": user.permissions,
            "data_scope": user.data_scopes,
            
            # 安全上下文
            "client_type": "web",
            "auth_method": "password+mfa" if user.mfa_enabled else "password",
        }
        
        # 生成令牌
        token = jwt.encode(
            payload,
            self.private_key,
            algorithm="HS256",
            headers={"kid": self.key_version}
        )
        
        return token
    
    def verify_token(self, token: str) -> dict:
        """验证令牌"""
        try:
            # 1. 解码并验证签名
            payload = jwt.decode(
                token,
                self.public_key,
                algorithms=["HS256"],
                audience="nesom-api",
                issuer="nesom-auth"
            )
            
            # 2. 检查令牌是否在黑名单中
            if self.is_token_revoked(payload["jti"]):
                raise jwt.InvalidTokenError("Token已被撤销")
            
            # 3. 检查密钥版本
            if payload.get("ver") != self.key_version:
                raise jwt.InvalidTokenError("密钥版本不匹配")
            
            # 4. 检查令牌类型
            if payload.get("typ") != "access":
                raise jwt.InvalidTokenError("令牌类型错误")
            
            return payload
            
        except jwt.ExpiredSignatureError:
            raise jwt.ExpiredSignatureError("令牌已过期")
        except jwt.InvalidTokenError as e:
            raise jwt.InvalidTokenError(f"令牌无效: {str(e)}")
```

class TokenRevocationService:
    def __init__(self):
        self.redis = get_redis_client()
        
    def revoke_token(self, jti: str, reason: str = None, ttl: int = None):
        """撤销令牌"""
        # 计算黑名单TTL（通常比令牌过期时间长一些）
        if ttl is None:
            ttl = 1800  # 30分钟，默认值
            
        # 存储到Redis黑名单
        key = f"token:blacklist:{jti}"
        self.redis.setex(
            key,
            ttl,
            json.dumps({
                "jti": jti,
                "revoked_at": datetime.utcnow().isoformat(),
                "reason": reason or "user_logout"
            })
        )
        
        # 发布撤销事件（用于集群同步）
        self.redis.publish("token:revoked", jti)
    
    def revoke_all_user_tokens(self, user_id: str, reason: str = "password_changed"):
        """撤销用户所有令牌"""
        # 获取用户所有活跃会话
        sessions = self.get_user_sessions(user_id)
        
        for session in sessions:
            self.revoke_token(session.jti, reason)
            
        # 记录审计日志
        self.log_revocation_event(user_id, "all", reason)
    
    def is_token_revoked(self, jti: str) -> bool:
        """检查令牌是否已被撤销"""
        key = f"token:blacklist:{jti}"
        return self.redis.exists(key) > 0
```

class SessionSecurityManager:
    def create_session(self, user: User, client_info: dict) -> Session:
        """创建新会话"""
        # 1. 生成唯一会话ID
        session_id = self.generate_session_id()
        
        # 2. 检查并发会话限制
        active_sessions = self.get_user_active_sessions(user.id)
        if len(active_sessions) >= self.MAX_CONCURRENT_SESSIONS:
            # 移除最旧的会话
            oldest_session = min(active_sessions, key=lambda s: s.created_at)
            self.revoke_session(oldest_session.id, "concurrent_limit")
        
        # 3. 创建会话记录
        session = Session(
            id=session_id,
            user_id=user.id,
            client_type=client_info.get("type", "web"),
            user_agent=client_info.get("user_agent"),
            ip_address=client_info.get("ip"),
            geo_location=client_info.get("geo"),
            device_fingerprint=client_info.get("fingerprint"),
            created_at=datetime.utcnow(),
            last_activity=datetime.utcnow(),
            expires_at=datetime.utcnow() + timedelta(hours=24)
        )
        
        # 4. 存储会话（Redis + 数据库）
        self.store_session(session)
        
        # 5. 生成令牌
        access_token = self.jwt_service.create_access_token(user, session_id)
        refresh_token = self.generate_refresh_token(session_id)
        
        return {
            "session": session,
            "access_token": access_token,
            "refresh_token": refresh_token
        }
    
    def validate_session(self, session_id: str, user_id: str) -> bool:
        """验证会话有效性"""
        # 1. 检查会话是否存在
        session = self.get_session(session_id)
        if not session:
            return False
            
        # 2. 检查用户匹配
        if session.user_id != user_id:
            self.revoke_session(session_id, "user_mismatch")
            return False
            
        # 3. 检查会话是否过期
        if session.expires_at < datetime.utcnow():
            self.revoke_session(session_id, "expired")
            return False
            
        # 4. 检查是否被撤销
        if session.revoked:
            return False
            
        # 5. 检查异常活动（可选）
        if self.detect_abnormal_activity(session):
            session.status = "abnormal"
            self.update_session(session)
            # 可以要求重新认证
            return False
            
        # 6. 更新最后活动时间（滑动过期）
        session.last_activity = datetime.utcnow()
        self.update_session(session)
        
        return True
```

from cryptography.hazmat.primitives.ciphers.aead import AESGCM
import base64
import os

class FieldEncryptionService:
    def __init__(self, key_provider):
        self.key_provider = key_provider
        
    def encrypt_field(self, plaintext: str, context: dict = None) -> str:
        """加密字段值"""
        if not plaintext:
            return plaintext
            
        # 1. 获取数据加密密钥
        dek = self.key_provider.get_data_key(context)
        
        # 2. 生成随机IV
        iv = os.urandom(12)
        
        # 3. 准备AAD（附加认证数据）
        aad = self._build_aad(context)
        
        # 4. 加密数据
        aesgcm = AESGCM(dek)
        ciphertext = aesgcm.encrypt(iv, plaintext.encode('utf-8'), aad)
        
        # 5. 组合结果：IV + 密文
        result = iv + ciphertext
        
        # 6. Base64编码
        return base64.b64encode(result).decode('utf-8')
    
    def decrypt_field(self, encrypted_data: str, context: dict = None) -> str:
        """解密字段值"""
        if not encrypted_data:
            return encrypted_data
            
        try:
            # 1. Base64解码
            data = base64.b64decode(encrypted_data)
            
            # 2. 分离IV和密文
            iv = data[:12]
            ciphertext = data[12:]
            
            # 3. 获取数据加密密钥
            dek = self.key_provider.get_data_key(context)
            
            # 4. 准备AAD
            aad = self._build_aad(context)
            
            # 5. 解密数据
            aesgcm = AESGCM(dek)
            plaintext = aesgcm.decrypt(iv, ciphertext, aad)
            
            return plaintext.decode('utf-8')
            
        except Exception as e:
            # 记录错误但不泄露细节
            logger.error(f"字段解密失败: {str(e)}")
            raise ValueError("数据解密失败")

        r'("phone":\s*)"[^"]*"': r'\1"***"',
        
        # MFA相关
        r'("mfa_code":\s*)"[^"]*"': r'\1"***"',
        r'("mfa_secret":\s*)"[^"]*"': r'\1"***"',
    }
    
    def sanitize_log(self, log_data: dict) -> dict:
        """脱敏日志数据"""
        import json
        import re
        
        # 转换为JSON字符串进行处理
        log_str = json.dumps(log_data, ensure_ascii=False)
        
        # 应用所有脱敏规则
        for pattern, replacement in self.SENSITIVE_PATTERNS.items():
            log_str = re.sub(pattern, replacement, log_str)
        
        # 转换回字典
        return json.loads(log_str)
