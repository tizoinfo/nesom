**版本**: 1.0  
**日期**: 2026-04-01  
**作者**: 智能体编排者

## 1. 设计概述

### 1.1 设计目标
巡检管理前端模块提供全面、高效、移动优先的巡检管理界面，支持：
- **计划管理**: 巡检计划的创建、编辑、排期管理
- **任务执行**: 巡检任务的分配、执行、结果提交
- **移动巡检**: 移动端现场巡检、数据采集、离线支持
- **路线规划**: 巡检路线可视化、优化、导航
- **报告分析**: 巡检报告查看、统计、数据可视化
- **实时监控**: 巡检进度实时监控、问题即时通知

### 1.2 设计原则
1. **移动优先**: 充分考虑现场巡检的移动端需求，支持离线操作
2. **实时性**: 任务状态实时更新，进度实时同步
3. **直观性**: 数据可视化，状态一目了然，地图集成
4. **一致性**: Web端和移动端体验一致，组件规范统一
5. **性能**: 大数据量下的流畅体验，移动端资源优化
6. **可访问性**: 支持离线操作、断点续传、冲突解决

### 1.3 技术约束
- **Web框架**: Vue 3.4 Composition API + `<script setup>`
- **移动端框架**: Vue 3 + Vant 4 + Capacitor (跨平台)
- **语言**: TypeScript 5.3 (严格模式)
- **Web UI库**: Element Plus 2.7 + 自定义主题
- **移动端UI库**: Vant 4 (轻量级移动端组件库)
- **状态管理**: Pinia (统一状态管理)
- **路由**: Vue Router 4 (Web), Vue Router + 移动端路由守卫
- **HTTP客户端**: Axios + 自定义封装 + 离线拦截器
- **图表**: ECharts 5 (Web), F2 (移动端轻量图表)
- **地图**: 百度地图/高德地图API (Web), 原生地图API (移动端)
- **WebSocket**: 原生WebSocket + 自动重连 (实时通知)
- **离线存储**: IndexedDB + LocalForage (移动端离线数据)
- **文件处理**: 文件分片上传、图片压缩、本地缓存

- 高级设置: 分配策略、检查要求、通知设置

**组件结构**:
```vue
<template>
  <div class="inspection-plan-editor">
    <!-- 基础信息 -->
    <PlanBasicInfo v-model="planData.basic" />
    
    <!-- 频率设置 -->
    <FrequencySelector v-model="planData.frequency" />
    
    <!-- 路线选择 -->
    <RouteSelector 
      v-model="planData.route" 
      :station-id="planData.stationId"
    />
    
    <!-- 模板选择 -->
    <TemplateSelector v-model="planData.template" />
    
    <!-- 高级设置 -->
    <PlanAdvancedSettings v-model="planData.settings" />
    
    <!-- 预览和提交 -->
    <PlanPreview :plan="planData" />
    <FormActions @submit="handleSubmit" @cancel="handleCancel" />
  </div>
</template>
```

**Props**:
```typescript
interface Props {
  planId?: string; // 编辑时传入计划ID
  stationId?: string; // 关联场站ID
  mode?: 'create' | 'edit' | 'copy'; // 模式
}
```

**状态管理**:
```typescript
const planStore = usePlanStore();

// 表单数据
const planData = ref<InspectionPlanForm>({
  basic: { /* 基础信息 */ },
  frequency: { /* 频率设置 */ },
  route: null,
  template: null,
  settings: { /* 高级设置 */ }
});

// 提交处理
const handleSubmit = async () => {
  const result = await planStore.savePlan(planData.value);
  if (result.success) {
    message.success('计划保存成功');
    router.push(`/inspection/plans/${result.data.id}`);
  }
};
```

```vue
        {{ statusMessage }}
      </div>
      
      <!-- 同步进度 -->
      <div v-if="isSyncing" class="sync-progress">
        <div class="progress-bar">
          <div
            class="progress-fill"
            :style="{ width: `${syncProgress}%` }"
          ></div>
        </div>
        <div class="progress-text">
          同步中: {{ syncProgress }}% ({{ syncedCount }}/{{ totalCount }})
        </div>
      </div>
      
      <!-- 冲突提示 -->
      <div v-if="hasConflicts" class="conflict-alert">
        <van-icon name="warning" />
        <span>发现 {{ conflictCount }} 个数据冲突需要解决</span>
        <van-button size="mini" @click="resolveConflicts">
          解决冲突
        </van-button>
      </div>
    </div>
    
    <!-- 操作按钮 -->
    <div class="action-buttons">
      <van-button
        v-if="isOffline && hasPendingChanges"
        size="small"
        @click="forceSync"
      >
        尝试同步
      </van-button>
      
      <van-button
        v-if="isOnline && hasPendingChanges"
        size="small"
        type="primary"
        @click="startSync"
      >
        立即同步
      </van-button>
    </div>
  </div>
</template>
```

**状态管理**:
