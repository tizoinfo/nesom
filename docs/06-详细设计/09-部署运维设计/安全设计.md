## 1. 设计概述

### 1.1 设计目标
构建NESOM系统部署运维层的全方位安全防护体系，确保运维操作的安全性、可靠性和合规性，包括：
- 保障运维基础设施和工具的访问安全
- 实现运维操作的细粒度权限控制和审计追溯
- 保护运维数据（配置、密钥、日志）的机密性和完整性
- 防范针对运维系统的攻击和滥用
- 满足监管合规要求和行业安全标准

### 1.2 设计原则
1. **最小权限原则**：运维人员仅拥有完成工作所需的最小权限
2. **职责分离原则**：关键运维操作需要多人协作完成
3. **操作透明原则**：所有运维操作可审计、可追溯、可验证
4. **纵深防御原则**：多层安全防护，单一防线失效不影响整体安全
5. **默认安全原则**：安全配置默认开启，需要时显式关闭
6. **持续验证原则**：定期验证安全控制的有效性

### 1.3 威胁模型分析
| 威胁类别 | 具体威胁 | 影响程度 | 发生概率 | 防护措施 |
|----------|----------|----------|----------|----------|
| 身份冒用 | 凭证泄露、会话劫持 | 高 | 中 | 多因素认证、会话管理 |
| 权限提升 | 越权操作、配置错误 | 高 | 中 | 最小权限、操作审批 |
| 数据泄露 | 配置泄露、日志暴露 | 高 | 中 | 加密存储、访问控制 |
| 操作滥用 | 恶意操作、误操作 | 高 | 中 | 操作审批、命令审查 |
| 供应链攻击 | 恶意镜像、依赖漏洞 | 中 | 低 | 镜像签名、依赖检查 |
| 审计绕过 | 日志篡改、记录缺失 | 中 | 低 | 不可变日志、完整性保护 |

### 1.4 安全责任矩阵

```python
class OperationsIdentityProvider:
    """运维专用身份提供者"""
    
    def __init__(self, config):
        self.config = config
        self.session_manager = SessionManager()
        self.risk_engine = RiskAssessmentEngine()
        
    def authenticate_operator(self, credentials, context):
        """运维人员认证"""
        # 1. 基础认证
        auth_result = self.basic_authentication(credentials)
        if not auth_result.success:
            return auth_result
        
        # 2. 风险评估
        risk_assessment = self.risk_engine.assess_login_risk(
            auth_result.user,
            context
        )
        
        if risk_assessment.risk_level == 'high':
            # 高风险登录需要额外验证
            return AuthenticationResult(
                success=False,
                requires_additional_factors=True,
                risk_level='high',
                challenge_methods=risk_assessment.challenge_methods
            )
        
        # 3. MFA验证
        if self.requires_mfa(auth_result.user, context):
            mfa_result = self.verify_mfa(credentials.mfa_token, auth_result.user)
            if not mfa_result.success:
                return mfa_result
        
        # 4. 环境验证（如IP白名单、工作时间）
        environment_check = self.check_environment_constraints(context)
        if not environment_check.allowed:
            return AuthenticationResult(
                success=False,
                reason=f"环境限制: {environment_check.reason}"
            )
        
        # 5. 创建安全会话
        session = self.session_manager.create_secure_session(
            user=auth_result.user,
            context=context,
            risk_level=risk_assessment.risk_level
        )
        
        return AuthenticationResult(
            success=True,
            user=auth_result.user,
            session_token=session.token,
            session_metadata=session.metadata
        )
    
    def requires_mfa(self, user, context):
        """判断是否需要MFA"""
        # 基于策略判断
        policies = self.get_mfa_policies(user.role, context.environment)
        
        for policy in policies:
            if self.policy_applies(policy, context):
                return policy.requires_mfa
        
        # 默认需要MFA
        return True
    
    def check_environment_constraints(self, context):
        """检查环境约束"""
        constraints = self.get_environment_constraints(context.environment)
        
        violations = []
        
        # IP地址检查
        if constraints.ip_whitelist:
            if context.ip_address not in constraints.ip_whitelist:
                violations.append(f"IP地址不在白名单内: {context.ip_address}")
        
        # 工作时间检查
        if constraints.working_hours:
            if not self.is_within_working_hours(context.timestamp, constraints.working_hours):
                violations.append("非工作时间访问")
        
        # 地理位置检查
        if constraints.geo_restrictions:
            location = self.geolocate_ip(context.ip_address)
            if location not in constraints.geo_restrictions:
                violations.append(f"地理位置限制: {location}")
        
        return EnvironmentCheckResult(
            allowed=len(violations) == 0,
            violations=violations
        )
```

#### 2.2.2 基于属性的访问控制（ABAC）
```python
class OperationsABACEngine:
    """运维ABAC引擎"""
    
    def check_operation_permission(self, request: OperationRequest) -> PermissionResult:
        """检查操作权限"""
        # 1. 获取用户属性
        user_attributes = self.get_user_attributes(request.user_id)
        
        # 2. 获取资源属性
        resource_attributes = self.get_resource_attributes(request.resource_type, request.resource_id)
        
        # 3. 获取环境属性
        environment_attributes = self.get_environment_attributes(request.environment_id)
        
        # 4. 获取操作属性
        operation_attributes = {
            'action': request.action,
            'severity': self.assess_operation_severity(request),
            'time': request.timestamp,
            'justification': request.justification
        }
        
        # 5. 评估策略
        policies = self.get_relevant_policies(
            user_attributes,
            resource_attributes,
            environment_attributes,
            operation_attributes
        )
        
        # 6. 应用策略
        decision = self.evaluate_policies(policies, request)
        
        # 7. 记录决策
        self.record_authorization_decision(request, decision)
        
        return decision
    
    def get_relevant_policies(self, user_attrs, resource_attrs, env_attrs, op_attrs):
        """获取相关策略"""
        relevant_policies = []
        
        for policy in self.policies:
            if self.policy_matches(policy, user_attrs, resource_attrs, env_attrs, op_attrs):
                relevant_policies.append(policy)
        
        # 按优先级排序
        relevant_policies.sort(key=lambda p: p.priority, reverse=True)
        
        return relevant_policies
    
    def policy_matches(self, policy, user_attrs, resource_attrs, env_attrs, op_attrs):
        """检查策略是否匹配"""
        # 检查用户条件
        if not self.attributes_match(policy.user_conditions, user_attrs):
            return False
        
        # 检查资源条件
        if not self.attributes_match(policy.resource_conditions, resource_attrs):
            return False
        
        # 检查环境条件
        if not self.attributes_match(policy.environment_conditions, env_attrs):
            return False
        
        # 检查操作条件
        if not self.attributes_match(policy.operation_conditions, op_attrs):
            return False
        
        return True
    
    def evaluate_policies(self, policies, request):
        """评估策略"""
        if not policies:
            # 默认拒绝
            return PermissionResult(
                allowed=False,
                reason="无适用策略，默认拒绝",
                obligations=[]
            )
        
        for policy in policies:
            if policy.effect == "allow":
                # 检查约束条件
                constraints_satisfied = self.check_constraints(policy.constraints, request)
                if constraints_satisfied:
                    return PermissionResult(
                        allowed=True,
                        reason=f"策略允许: {policy.id}",
                        obligations=policy.obligations
                    )
            else:  # deny
                return PermissionResult(
                    allowed=False,
                    reason=f"策略拒绝: {policy.id}",
                    obligations=[]
                )
        
        # 没有明确允许的策略
        return PermissionResult(
            allowed=False,
            reason="所有适用策略评估后拒绝",
            obligations=[]
        )
```

#### 3.1.1 多级审批机制
```python
class OperationsApprovalWorkflow:
    """运维操作审批工作流"""
    
    def __init__(self, workflow_config):
        self.config = workflow_config
        self.approval_store = ApprovalStore()
        
    def create_approval_request(self, operation, context):
        """创建审批请求"""
        # 1. 确定审批要求
        approval_requirements = self.determine_approval_requirements(operation, context)
        
        # 2. 选择审批人
        approvers = self.select_approvers(approval_requirements, context)
        
        # 3. 创建审批请求
        approval_request = ApprovalRequest(
            operation_id=operation.id,
            operation_type=operation.type,
            requester=context.user_id,
            approvers=approvers,
            requirements=approval_requirements,
            context=context,
            created_at=datetime.now()
        )
        
        # 4. 存储审批请求
        self.approval_store.save_approval_request(approval_request)
        
        # 5. 通知审批人
        self.notify_approvers(approval_request)
        
        return approval_request
    
    def determine_approval_requirements(self, operation, context):
        """确定审批要求"""
        requirements = {
            'required': False,
            'level': 'standard',
            'quorum': 1,
            'timeout': '24小时',
            'escalation_path': []
        }
        
        # 基于操作类型确定审批要求
        if operation.type in ['deploy', 'rollback']:
            if context.environment == 'production':
                requirements['required'] = True
                requirements['level'] = 'high'
                requirements['quorum'] = 2
                requirements['timeout'] = '2小时'
                requirements['escalation_path'] = ['team_lead', 'sre_manager', 'cto']
        
        elif operation.type in ['backup_restore', 'database_migration']:
            requirements['required'] = True
            requirements['level'] = 'critical'
            requirements['quorum'] = 3
            requirements['timeout'] = '4小时'
            requirements['escalation_path'] = ['dba_lead', 'security_lead', 'cto']
        
        elif operation.type in ['permission_change', 'config_update']:
            requirements['required'] = True
            requirements['level'] = 'medium'
            requirements['quorum'] = 1
            requirements['timeout'] = '8小时'
            requirements['escalation_path'] = ['security_team']
        
        # 基于风险评分调整
        risk_score = self.assess_operation_risk(operation, context)
        if risk_score > 80:
            requirements['level'] = 'critical'
            requirements['quorum'] += 1
        
        return requirements
    
    def select_approvers(self, requirements, context):
        """选择审批人"""
        approvers = []
        
        # 基于角色选择审批人
        required_roles = self.get_required_roles(requirements['level'])
        
        for role in required_roles:
            # 从该角色中选择合适的审批人
            candidates = self.get_users_by_role(role)
            
            # 过滤掉请求者和冲突人员
            filtered = [
                user for user in candidates
                if user.id != context.user_id
                and not self.has_conflict_of_interest(user, context)
            ]
            
            if filtered:
                # 选择最合适的审批人（基于可用性、专业领域等）
                selected = self.select_most_appropriate_approver(filtered, context)
                approvers.append(selected)
        
        # 确保达到法定人数
        if len(approvers) < requirements['quorum']:
            # 从escalation_path中选择额外审批人
            additional = self.select_from_escalation_path(
                requirements['escalation_path'],
                requirements['quorum'] - len(approvers),
                context
            )
            approvers.extend(additional)
        
        return approvers
    
    def process_approval_response(self, approval_id, approver_id, decision, comments):
        """处理审批响应"""
        approval = self.approval_store.get_approval_request(approval_id)
        
        # 记录审批决定
        approval.record_response(
            approver_id=approver_id,
            decision=decision,
            comments=comments,
            timestamp=datetime.now()
        )
        
        # 检查是否达到法定人数
        if approval.reached_quorum():
            if approval.all_approved():
                # 所有审批人同意
                approval.status = 'approved'
                self.execute_approved_operation(approval.operation_id)
            else:
                # 有审批人拒绝
                approval.status = 'rejected'
                self.notify_requester_rejection(approval)
        
        elif approval.is_timed_out():
            # 审批超时，升级
            approval.status = 'escalated'
            self.escalate_approval(approval)
        
        # 更新审批状态
        self.approval_store.update_approval_request(approval)
        
        return approval
```

### 3.2 命令执行安全

#### 3.2.1 命令白名单和黑名单
```python
class CommandSecurityFilter:
    """命令安全过滤器"""
    
    def __init__(self):
        self.whitelist = self.load_whitelist()
        self.blacklist = self.load_blacklist()
        self.patterns = self.load_dangerous_patterns()
        
    def validate_command(self, command, context):
        """验证命令安全性"""
        validation_result = {
            'allowed': False,
            'reasons': [],
            'suggestions': [],
            'risk_level': 'unknown'
        }
        
        # 1. 检查黑名单
        if self.is_blacklisted(command):
            validation_result['reasons'].append("命令在黑名单中")
            validation_result['risk_level'] = 'critical'
            return validation_result
        
        # 2. 检查白名单
        if self.is_whitelisted(command):
            validation_result['allowed'] = True
            validation_result['risk_level'] = 'low'
        else:
            # 不在白名单中，需要进一步检查
            validation_result['reasons'].append("命令不在白名单中")
            validation_result['risk_level'] = 'high'
        
        # 3. 检查危险模式
        dangerous_patterns = self.find_dangerous_patterns(command)
        if dangerous_patterns:
            validation_result['reasons'].extend([
                f"检测到危险模式: {pattern}" for pattern in dangerous_patterns
            ])
            validation_result['risk_level'] = max(
                validation_result['risk_level'],
                'high'
            )
            validation_result['allowed'] = False
        
        # 4. 检查上下文相关性
        if not self.is_context_appropriate(command, context):
            validation_result['reasons'].append("命令与上下文不匹配")
            validation_result['risk_level'] = 'medium'
            validation_result['allowed'] = False
        
        # 5. 生成建议
        if not validation_result['allowed']:
            validation_result['suggestions'] = self.generate_suggestions(
                command,
                validation_result['reasons']
            )
        
        return validation_result
    
    def is_whitelisted(self, command):
        """检查命令是否在白名单中"""
        # 精确匹配
        if command in self.whitelist['exact']:
            return True
        
        # 模式匹配
        for pattern in self.whitelist['patterns']:
            if re.match(pattern, command):
                return True
        
        # 参数化命令检查
        base_command = self.extract_base_command(command)
        if base_command in self.whitelist['base_commands']:
            # 检查参数是否允许
            return self.are_parameters_allowed(command, base_command)
        
        return False
    
    def find_dangerous_patterns(self, command):
        """查找危险模式"""
        dangerous = []
        
        for category, patterns in self.patterns.items():
            for pattern in patterns:
                if re.search(pattern, command):
                    dangerous.append(f"{category}: {pattern}")
        
        return dangerous
    
    def generate_suggestions(self, command, reasons):
        """生成安全建议"""
        suggestions = []
        
        if "命令不在白名单中" in reasons:
            # 建议使用白名单中的替代命令
            alternatives = self.find_alternatives(command)
            if alternatives:
                suggestions.append(f"建议使用以下安全命令: {', '.join(alternatives)}")
            else:
                suggestions.append("请联系安全团队将此命令添加到白名单")
        
        if any("危险模式" in reason for reason in reasons):
            suggestions.append("请重新评估命令的必要性，或联系安全团队获取指导")
        
        if "命令与上下文不匹配" in reasons:
            suggestions.append("请确认命令在正确的上下文中执行")
        
        return suggestions
```

```python
class OperationsKeyManagement:
    """运维密钥管理"""
    
    def __init__(self, hsm_client, key_store):
        self.hsm = hsm_client
        self.key_store = key_store
        self.key_cache = {}
        
    def encrypt_secret(self, secret_type, secret_value, context):
        """加密敏感数据"""
        # 1. 获取或生成数据加密密钥（DEK）
        dek_id, dek = self.get_or_create_dek(secret_type, context)
        
        # 2. 使用DEK加密数据
        cipher = AES.new(dek, AES.MODE_GCM)
        ciphertext, tag = cipher.encrypt_and_digest(
            secret_value.encode('utf-8')
        )
        
        # 3. 使用主密钥加密DEK
        encrypted_dek = self.encrypt_dek_with_master_key(dek)
        
        # 4. 存储加密数据
        encrypted_secret = {
            'ciphertext': b64encode(ciphertext).decode('ascii'),
            'tag': b64encode(tag).decode('ascii'),
            'nonce': b64encode(cipher.nonce).decode('ascii'),
            'encrypted_dek': b64encode(encrypted_dek).decode('ascii'),
            'dek_id': dek_id,
            'algorithm': 'AES-256-GCM',
            'encrypted_at': datetime.now().isoformat(),
            'context': context
        }
        
        return encrypted_secret
    
    def decrypt_secret(self, encrypted_secret, context):
        """解密敏感数据"""
        # 1. 验证访问权限
        if not self.has_decrypt_permission(encrypted_secret, context):
            raise PermissionError("无解密权限")
        
        # 2. 解密DEK
        encrypted_dek = b64decode(encrypted_secret['encrypted_dek'])
        dek = self.decrypt_dek_with_master_key(encrypted_dek)
        
        # 3. 解密数据
        ciphertext = b64decode(encrypted_secret['ciphertext'])
        nonce = b64decode(encrypted_secret['nonce'])
        tag = b64decode(encrypted_secret['tag'])
        
        cipher = AES.new(dek, AES.MODE_GCM, nonce=nonce)
        plaintext = cipher.decrypt_and_verify(ciphertext, tag)
        
        # 4. 记录解密审计日志
        self.record_decryption_audit(encrypted_secret, context)
        
        return plaintext.decode('utf-8')
    
    def get_or_create_dek(self, secret_type, context):
        """获取或创建数据加密密钥"""
        # 基于secret_type和context生成密钥ID
        key_id = self.generate_key_id(secret_type, context)
        
        # 检查缓存
        if key_id in self.key_cache:
            cached = self.key_cache[key_id]
            if not cached.is_expired():
                return key_id, cached.key
        
        # 检查密钥存储
        stored_key = self.key_store.get_key(key_id)
        if stored_key:
            key = stored_key
        else:
            # 生成新密钥
            key = self.generate_new_key()
            
            # 存储密钥
            self.key_store.store_key(key_id, key, context)
        
        # 更新缓存
        self.key_cache[key_id] = CachedKey(
            key=key,
            expires_at=datetime.now() + timedelta(hours=1)
        )
        
        return key_id, key
    
    def rotate_keys(self, key_type=None, force=False):
        """轮换加密密钥"""
        keys_to_rotate = self.key_store.get_keys_for_rotation(key_type, force)
        
        rotation_report = {
            'started_at': datetime.now(),
            'total_keys': len(keys_to_rotate),
            'successful': 0,
            'failed': 0,
            'details': []
        }
        
        for key_info in keys_to_rotate:
            try:
                # 1. 生成新密钥
                new_key = self.generate_new_key()
                
                # 2. 重新加密使用此密钥加密的所有数据
                reencrypted_count = self.reencrypt_data(
                    key_info.key_id,
                    new_key
                )
                
                # 3. 更新密钥存储
                self.key_store.update_key(
                    key_info.key_id,
                    new_key,
                    key_info.context
                )
                
                # 4. 记录成功
                rotation_report['successful'] += 1
                rotation_report['details'].append({
                    'key_id': key_info.key_id,
                    'status': 'success',
                    'reencrypted_count': reencrypted_count
                })
                
            except Exception as e:
                rotation_report['failed'] += 1
                rotation_report['details'].append({
                    'key_id': key_info.key_id,
                    'status': 'failed',
                    'error': str(e)
                })
        
        rotation_report['completed_at'] = datetime.now()
        
        return rotation_report
```
