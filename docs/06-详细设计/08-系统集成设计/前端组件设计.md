## 1. 设计概述

### 1.1 设计目标
设计NESOM系统集成管理的前端组件，提供直观、高效、可操作的集成监控、配置和管理界面，包括：
- 集成健康状态可视化监控仪表盘
- 跨模块数据流实时追踪和展示
- 集成配置和规则管理界面
- 故障诊断和问题排查工具
- 集成性能分析和优化建议

### 1.2 设计原则
1. **可视化优先**：复杂集成状态通过图表和图形直观展示
2. **实时性**：关键指标和数据流实时更新，支持自动刷新
3. **可操作性**：发现问题后可直接在界面上进行操作干预
4. **上下文关联**：展示数据时提供完整的上下文信息
5. **响应式设计**：支持桌面、平板、移动端多设备访问
6. **无障碍访问**：符合WCAG 2.1 AA标准，支持辅助技术

### 1.3 技术栈选择
- **前端框架**: Vue 3.4 + Composition API + TypeScript
- **UI组件库**: Element Plus 2.3 + 自定义主题
- **状态管理**: Pinia 2.1 (替代Vuex)
- **图表库**: ECharts 5.4 (高性能图表) + D3.js 7.8 (复杂可视化)
- **地图组件**: Mapbox GL JS 2.15 (地理位置可视化)
- **实时通信**: Socket.IO 4.7 (WebSocket实时数据)
- **构建工具**: Vite 5.0 + Rollup
- **代码规范**: ESLint + Prettier + TypeScript严格模式

### 1.4 组件分类
| 组件类别 | 功能定位 | 使用角色 | 典型场景 |
|----------|----------|----------|----------|
| 监控仪表盘 | 集成健康状态概览 | 运维人员、管理员 | 日常监控、快速巡检 |
| 数据流追踪 | 跨模块数据流动展示 | 开发人员、架构师 | 问题排查、流程分析 |
| 配置管理 | 集成规则和配置管理 | 管理员、配置员 | 规则调整、配置更新 |
| 故障诊断 | 问题诊断和修复工具 | 运维人员、开发人员 | 故障处理、性能优化 |
| 报表分析 | 集成性能数据分析 | 管理员、架构师 | 容量规划、优化决策 |

```vue
<script setup lang="ts">
import { computed } from 'vue'

interface Props {
  healthData: IntegrationHealthData
}

const props = defineProps<Props>()

const overallStatus = computed(() => {
  const { components } = props.healthData
  if (components.some(c => c.status === 'unhealthy')) return 'unhealthy'
  if (components.some(c => c.status === 'degraded')) return 'degraded'
  return 'healthy'
})

const healthScore = computed(() => {
  const { components } = props.healthData
  const healthyCount = components.filter(c => c.status === 'healthy').length
  return Math.round((healthyCount / components.length) * 10000) / 100
})

const statusDistribution = computed(() => {
  const statusCounts = {
    healthy: 0,
    degraded: 0,
    unhealthy: 0,
    unknown: 0
  }
  
  props.healthData.components.forEach(c => {
    statusCounts[c.status as keyof typeof statusCounts]++
  })
  
  const total = props.healthData.components.length
  return Object.entries(statusCounts).map(([status, count]) => ({
    status,
    name: statusText[status as keyof typeof statusText],
    count,
    percentage: Math.round((count / total) * 10000) / 100
  }))
})

const statusText = {
  healthy: '健康',
  degraded: '降级',
  unhealthy: '异常',
  unknown: '未知'
}
</script>
```

### 2.2 数据流追踪器 (DataFlowTracker)

#### 2.2.1 组件概述
**组件名称**: `DataFlowTracker`  
**文件路径**: `frontend/src/views/integration/tracker/DataFlowTracker.vue`  
**功能描述**: 可视化展示跨模块数据流动路径和状态  
**数据来源**: `/api/v1/integration/events`, WebSocket实时数据  
**交互特性**: 支持拖拽、缩放、点击查看详情

#### 2.2.2 组件设计要点
1. **模块节点图**: 7个模块作为节点，显示当前状态和负载
2. **数据流连线**: 实时显示数据流动方向和速度
3. **事件详情面板**: 点击节点或连线显示详细信息
4. **时间线控制**: 可回溯查看历史数据流动
5. **过滤器**: 按事件类型、状态、时间范围过滤

```typescript
        this.errors.health = error

        this.loading.health = false
      }
```

### 5.1 WebSocket连接管理
```typescript
// frontend/src/utils/websocket.ts
class IntegrationWebSocket {
  private socket: Socket | null = null
  private reconnectAttempts = 0
  private maxReconnectAttempts = 5
  private reconnectDelay = 1000
  private subscribers: Map<string, Function[]> = new Map()
  
  constructor() {
    this.connect()
  }
  
  connect() {
    const token = localStorage.getItem('access_token')
    this.socket = io(import.meta.env.VITE_WS_URL + '/integration', {
      auth: { token },
      transports: ['websocket', 'polling'],
      reconnection: true,
      reconnectionAttempts: this.maxReconnectAttempts,
      reconnectionDelay: this.reconnectDelay
    })
    
    this.setupEventHandlers()
  }
  
  private setupEventHandlers() {
    if (!this.socket) return
    
    this.socket.on('connect', () => {
      console.log('WebSocket连接成功')
      this.reconnectAttempts = 0
      this.notifySubscribers('connect', {})
    })
    
    this.socket.on('disconnect', (reason) => {
      console.log('WebSocket断开连接:', reason)
      this.notifySubscribers('disconnect', { reason })
      
      // 尝试重连
      if (this.reconnectAttempts < this.maxReconnectAttempts) {
        setTimeout(() => {
          this.reconnectAttempts++
          this.connect()
        }, this.reconnectDelay * Math.pow(2, this.reconnectAttempts))
      }
    })
    
    this.socket.on('metrics', (data) => {
      this.notifySubscribers('metrics', data)
    })
    
    this.socket.on('alert', (alert) => {
      this.notifySubscribers('alert', alert)
    })
    
    this.socket.on('event', (event) => {
      this.notifySubscribers('event', event)
    })
    
    this.socket.on('sync_progress', (progress) => {
      this.notifySubscribers('sync_progress', progress)
    })
  }
  
  subscribe(event: string, callback: Function) {
    if (!this.subscribers.has(event)) {
      this.subscribers.set(event, [])
    }
    this.subscribers.get(event)!.push(callback)
    
    // 返回取消订阅函数
    return () => {
      const callbacks = this.subscribers.get(event)
      if (callbacks) {
        const index = callbacks.indexOf(callback)
        if (index > -1) {
          callbacks.splice(index, 1)
        }
      }
    }
  }
  
  private notifySubscribers(event: string, data: any) {
    const callbacks = this.subscribers.get(event)
    if (callbacks) {
      callbacks.forEach(callback => {
        try {
          callback(data)
        } catch (error) {
          console.error(`事件${event}回调执行失败:`, error)
        }
      })
    }
  }
  
  emit(event: string, data: any) {
    if (this.socket?.connected) {
      this.socket.emit(event, data)
    } else {
      console.warn('WebSocket未连接，无法发送事件:', event)
    }
  }
  
  disconnect() {
    if (this.socket) {
      this.socket.disconnect()
      this.socket = null
    }
  }
}

export const integrationWS = new IntegrationWebSocket()
```

```typescript
// 为复杂组件添加键盘导航
function setupKeyboardNavigation(element: HTMLElement) {
  element.addEventListener('keydown', (event) => {
    const items = Array.from(
      element.querySelectorAll('[role="gridcell"], [role="listitem"]')
    )
    const currentIndex = items.findIndex(item => 
      item === document.activeElement || item.contains(document.activeElement)
    )
    
    switch (event.key) {
      case 'ArrowRight':
      case 'ArrowDown':
        event.preventDefault()
        if (currentIndex < items.length - 1) {
          (items[currentIndex + 1] as HTMLElement).focus()
        }
        break
        
      case 'ArrowLeft':
      case 'ArrowUp':
        event.preventDefault()
        if (currentIndex > 0) {
          (items[currentIndex - 1] as HTMLElement).focus()
        }
        break
        
      case 'Home':
        event.preventDefault()
        if (items.length > 0) {
          (items[0] as HTMLElement).focus()
        }
        break
        
      case 'End':
        event.preventDefault()
        if (items.length > 0) {
          (items[items.length - 1] as HTMLElement).focus()
        }
        break
    }
  })
}
```
